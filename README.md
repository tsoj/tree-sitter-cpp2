# tree-sitter-cpp2

To develop first run:
```bash
npm install
```

To build the parser:
```bash
tree-sitter generate
```

To run tests:
```bash
tree-sitter test
```

To run more extensive tests run:
```bash
# some/path could for example point to the cppfront regression tests directory
./test_on_cpp2_files.sh some/path
```
